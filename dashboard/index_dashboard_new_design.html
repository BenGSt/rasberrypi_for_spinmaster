SpinMaster Dashboard

SpinMaster is a machine for the separation of chiral molecule enantiomers, based on selective spin crystallization technology.

An undergraduate project by Ben Steinberg, Yoav Silberstein, and Daniel Fuchs, with the guidance of Omer Shoseyov, Dr. Shira Yochelis, and Prof. Yossi Paltiel.

Department of applied physics, the Hebrew University, Jerusalem, Isarel, 2022.


Sorce code and documentaion can be found at our github repo.
